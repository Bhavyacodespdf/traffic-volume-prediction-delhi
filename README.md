# Traffic Volume Prediction using Machine Learning

This project predicts traffic congestion and traffic volume in Delhi using
machine learning techniques applied to traffic, road network, and weather data.

## Project Status
- Data preprocessing completed
- Cleaned dataset prepared
- Model training in progress

## Tech Stack
- Python
- Pandas, GeoPandas
- Scikit-learn, XGBoost
- OpenStreetMap
- OpenWeather API
- Flask
